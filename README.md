# Portfolio Demo

Welcome to my personal portfolio website! This project showcases my skills, projects, and contact information in a modern, interactive, and responsive layout.

🔗 **Live Website:** [Portfolio Demo](https://kinetic-conflict.github.io/Portfolio_Demo/)

---

## Features

- **Responsive Design:** Works seamlessly on desktops, tablets, and mobile devices.
- **Project Showcase:** Highlights my projects with images, descriptions, and links.
- **Interactive Elements:** Smooth animations and transitions for an engaging user experience.
- **Contact Section:** Easily reach out via email or social links.
- **Clean & Modern Layout:** Minimalistic design focused on clarity and readability.

---

## Technologies Used

- **HTML5** – Structuring content for the web.
- **CSS3** – Styling and responsive design.
- **JavaScript** – Adding interactivity and animations.
- **GitHub Pages** – Hosting the live website.
